API Documentation

All endpoints return JSON. Base URL: http://localhost:5000/api

Authentication

No API key required for local use. For remote access, set API_KEY in your configuration and pass it as a header:

Header 
Authorization: Bearer YOUR_API_KEY

Privacy Score

GET /api/score Get privacy DNA score

Returns the current privacy DNA score and breakdown.

Response

JSON 
{
  "score": 72,
  "breakdown": {
    "broker_exposure": 35,
    "data_depth": 15,
    "breach_impact": 12,
    "social_exposure": 10
  },
  "grade": "B",
  "last_updated": "2024-01-15T14:30:00Z"
}

Code Examples

curl http://localhost:5000/api/score
import requests

response = requests.get("http://localhost:5000/api/score")
data = response.json()
print(f"Privacy Score: {data['score']}/100")
const response = await fetch('http://localhost:5000/api/score');
const data = await response.json();
console.log(`Privacy Score: ${data.score}/100`);

Scanning

POST /api/scan/start Start a new scan

Parameters

FieldTypeRequiredDescription
profile_idstringNoProfile to scan. Default: "all"
categoriesarrayNoBroker categories to scan

Response

{
  "scan_id": "scan_abc123",
  "status": "started",
  "total_brokers": 812
}
GET /api/scan/{scan_id}/status Get scan progress

Response

{
  "scan_id": "scan_abc123",
  "status": "running",
  "progress": 45,
  "brokers_checked": 365,
  "found": 12,
  "status_text": "Scanning B2B brokers..."
}

Displacement Mode

POST /api/displacement/activate Activate emergency lockdown

Activates displacement mode — accelerates all removals, sends credit freeze reminders.

POST /api/displacement/deactivate Deactivate emergency lockdown

Deactivates displacement mode and returns to normal protection levels.

Reports & Data

POST /api/reports/generate Generate a report

Body

{
  "type": "full",          // full | summary | family | category
  "format": "pdf",         // pdf | csv | json
  "date_from": "2024-01-01",
  "date_to": "2024-01-31"
}
GET /api/data/export Export all user data

Downloads a complete JSON export of all profiles, scan results, and opt-out history.

DELETE /api/data/all Delete all data

⚠️ Permanently deletes all data. This action cannot be undone.
